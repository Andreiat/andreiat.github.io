exec([string])¶

Возвращает массив найденных элементов в строке

var a = /\d/g;

a.exec('kj5k3');
// ['5']

a.exec('kj5k3');
// ['3']
test(string)¶

Возвращает булево, есть ли совпадение

var a = /\d/;

a.test('qw');
// false

a.test('123');
// true
