jQuery¶

class jQuery(selector[, context=document])¶
class jQuery(elts)
class jQuery(html[, props])
class jQuery(f)

  • selector - селектор для выбора, Селекторы

  • elts - эле­мен­т до­ку­мен­та, мас­сив или объ­ект, по­доб­ный мас­си­ву (NodeList или дру­гой объ­ект jQuery, со­дер­жа­щим эле­мен­ты до­ку­мен­та).

  • html - строка для син­так­си­че­ского ана­лиза

  • f - функ­ция, ко­то­рая долж­на быть вы­зва­на по­сле то­го, как до­ку­мент бу­дет за­гру­жен и ста­нет дос­ту­пен для вы­пол­не­ния опе­ра­ций с ним.

    Ес­ли до­ку­мент уже го­тов к вы­пол­не­нию опе­ра­ций, функ­ция f бу­дет вы­зва­на не­мед­лен­но, как ме­тод объ­ек­та document.

Элемент документа.

var jq_elem = $("body");
attr()¶

Задает атрибуты

jq_elem.attr("src", "image.png");
click()¶

Добавляет обработчик клика

jq_elem.click(function() {});
css()¶

Задает css стили

jq_elem.css("color", "red");
