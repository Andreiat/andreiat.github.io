dropable¶

Создание области падания элементов (например, у вас есть корзина товаров в интернет магазине, куда будут падать элементы, перенесённые в необходимую область мышкой)

dropable([methodName, option, value])
dropable([methodName, param_obj])
dropable([param_obj])
$('#dropable').dropable({
});

  • methodName

    • destroy - Полностью удаляет функциональность взаимодействия Droppable из эле­мента

    • disable - Временно отключает функциональность взаимодействия Droppable для базового элемента

    • enable - Включает ранее отключенную функциональность взаимодействия Droppable для базового элемента

    • option - Позволяет получить или изменить значение одной или нескольких опций

  • param_obj

    • accept - Сужает множество перемещаемых элементов, на которые будет реагировать прини­мающий элемент. Значение по умолчанию — *; ему соответствует любой элемент

    • activate - Происходит, когда пользователь начинает перетаскивать перемещаемый элемент

    • activeClass - Определяет класс, который будет присваиваться в ответ на событие activate и уда­ляться В ответ на СОбытие deactivate

    • create - Происходит в момент применения взаимодействия Draggable к элементу

    • deactivate - Происходит, когда пользователь прекращает перетаскивать перемещаемый элемент

    • disabled - Если эта опция равна true, то функциональность взаимодействия Droppable первона­чально отключена. Значение по умолчанию — false

    • drop - Происходит, когда пользователь оставляет перемещаемый элемент на принимающем элементе

    • fit - Перетаскиваемый элемент должен полностью находиться в области принимающего элемента

    • helper

    • hoverClass - Определяет класс, который будет присваиваться в ответ на событие over и удалятьсяв ответ на событие out

    • intersect - Перетаскиваемый элемент должен перекрываться с принимающим элементом по крайней мере наполовину. Это значение используется по умолчанию

    • out - Происходит, когда пользователь перетаскивает перемещаемый элемент за пределы принимающего элемента

    • over - Происходит, когда пользователь перетаскивает перемещаемый элемент над прини­мающим элементом (но при условии, что кнопка мыши еще не была отпущена)

    • pointer - Указатель мыши должен находиться в области принимающего элемента, независимо от того, где именно перетаскиваемый элемент был захвачен пользователем

    • scope

    • snap

    • snapMode

    • snapTolerance

    • tolerance - Определяет минимальную степень перекрывания, при которой происходит событие over

    • touch - Означает любую степень перекрывания перетаскиваемого и принимающего элементов
