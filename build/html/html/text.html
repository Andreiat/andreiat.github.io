b¶

Выделяет текст жирным

<b>жирный текст</a>

br¶

Перенос строки

<br />

h1…h6¶

Зоголовки, блочный элемент

<h1></h1>
...
<h6></h6>

hr¶

Горизонтальная линия, блочный элемент

<hr />

i¶

Выделяет текст курсивым

<i>курсивый текст</i>

mark¶

Примечание

HTML5

Выделение части документа, по причине ее вероятной значимости для того, чем в данный момент занят пользователь (напр.: пришли по ссылке из поисковика - подсветили нужные слова).

<mark>найденное слово</mark>

p¶

Блочный элемент, задающий абзац текста

<p></p>

span¶

Строчный элемент, используется для выделения части текста

<span></span>

sub¶

Нижний индекс для текста

<sub>индекс</sub>

sup¶

Верхний индекс для текста, стпень

<sup>степень</sup>
