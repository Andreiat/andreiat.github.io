HTML¶

HTML - декларативный язык программирования. Используемый для разметки web-страниц - html-документов. Web-страница отобржается в браузере (chrome, firefox, opera).

HTML документ состоит из тегов. Тег - элемент разметки, который может иметь атрибуты. Внутри тега может находиться обычный текст или другие теги.

<div></div>

<!-- тег с атрибутом -->
<тег атрибут=значение>
    <!--тут содержимое элемента-->

    <!--простой тег без атрибута-->
    <тег2>
        обычный текст
    </тег>

    <!--обычный текст-->
    другой текст

    <!--закрывающий тег, т.е. конец элемента-->
</тег>

<!DOCTYPE>

<html>

   <head>
       <meta charset="utf-8" />
       <title>Title</title>
   </head>

   <body></body>

</html>

Примечание

HTML5

Внедрение объекта на страницу

  • height - высота элемента

  • src - ссылка на ресурс

  • type - тип ресурса

  • width - ширина элемента

<embed src="ilnurgi.mov" width="240" height="320" type="video/quicktime"
