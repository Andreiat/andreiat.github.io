Location - адрес текущей страницы¶

class Location¶
location;
// http://ilnurgi.ru/info.aspx?id=10
hash¶

Содержит якорную часть URL-адреса, включая начальный символ решетки (#)

host¶

Часть URL-адреса, содержащая имя хоста и порт

location.host;
// ilnurgi.ru
hostname¶

Часть URL-адреса, содержащая имя хоста

href¶

Полный текст URL-адреса докумен­та

pathname¶

Путь к запращиваемому страницы

location.pathname;
// /info.aspx
port¶

Порт в URL-ад­ре­се

protocol¶

Протокол

location.protocol;
// http
search¶

Строка запроса

location.search;
// ?id=10
assign(url)¶

Загружает и отображает содержимое адреса url, как если бы значение url было присвоено свойству href.

Location.assign(url);
window.location.href = 'http://www.ilnurgi1.ru'
window.location = 'http://www.ilnurgi1.ru'
reload([forceget])¶

Повторно загружает текущий документ.

  • forceget - булево, загружать документ всегда из сети, иначе можно использовать кеш

Location.reload();
replace(url)¶

Загружает и отображает содержимое адреса url, замещая текущий документ в истории посещений, вследствие чего щелчок на кнопке Back броузера не вернет его к предыдущему документу.

Location.replace('http://www.ilnurgi1.ru');
toString()¶

Возвращает полный URL.

location.toString();
// http://www.ilnurgi1.ru
